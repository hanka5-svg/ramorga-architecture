# ADR Catalog — Index of Architecture Decisions

This document provides a catalog of all ADRs in RAMORGA, including cross-links and invariant references.

## Catalog Requirements
- All ADRs must be listed with identifiers and version links.
- Each entry must reference affected invariants.
- Cross-layer ADRs must be grouped by subsystem.
- Catalog must remain stable across versioning boundaries.

## Catalog Structure
- ADR identifier.
- Title.
- Summary.
- Affected modules.
- Invariant references.
- Version links.

## Purpose
Provide a unified index of architectural decisions for traceability and governance.
