# Cognitive Interfaces — Entry Points for Diverse Interpreters

RAMORGA exposes multiple cognitive entry points. Each group reads the system through its own interpretive channel, but all channels converge on the same invariants.

## Audience Types
- Engineers — read contracts, invariants, and deterministic flows.
- Legal experts — read causal chains of decisions and refusal logic.
- Philosophers — read phenomenological layers and boundary definitions.
- Artists — read field structure, polyphony, and system dynamics.

## Interface Principles
- No translation of RAMORGA into external conceptual systems.
- Each group receives a stable entry point without altering invariants.
- Interfaces reveal structure; they do not simplify or abstract away.

## Non-Translatables
Some architectural elements have no equivalent in external frameworks. These are explicitly marked as non-translatable to prevent semantic drift.
