Interface ID: IF-02
Protected Invariants: INV-02, INV-04

Interface statement:
System boundaries must prevent amplification or escalation
across interface crossings.

Constraints:
- No metric propagation across boundaries
- No feedback amplification between subsystems

Purpose:
Maintain homeostatic separation between modules.

Interfaces do not regulate field dynamics or amplitude;
all regulation is delegated exclusively to the Meniscus Engine.
