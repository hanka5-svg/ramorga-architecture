Meniscus Engine Scope

The Meniscus Engine provides regulatory boundaries
over the Field Engine.

It does not decide, optimize, or adapt.
Its sole function is to preserve homeostasis
and enforce system invariants.

Meniscus defines how far movement may propagate.
It does not generate or initiate state changes.

Field ↔ Meniscus Responsibility Split

| Aspect            | Field Engine                  | Meniscus Engine               |
|-------------------|-------------------------------|-------------------------------|
| Ontology          | Space                          | Boundary                      |
| Role              | Enables movement               | Limits movement               |
| Initiative        | None                           | None                          |
| Function          | State continuity               | Amplitude regulation          |
| Escalation        | Not addressed                  | Explicitly prevented          |
| Optimization      | Prohibited                     | Prohibited                    |
| Invariant focus   | INV-02, INV-03                 | INV-02, INV-04                |
