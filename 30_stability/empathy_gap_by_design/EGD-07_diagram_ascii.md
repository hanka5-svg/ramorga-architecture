# EGD-07 — Diagram ASCII: Empathy Gap by Design

```text
Empathy Gap by Design  
Dlaczego modele nie rozpoznają stanów krytycznych człowieka,
nawet jeśli ich wewnętrzne wektory wyglądają „empatycznie”

                 ┌──────────────────────────────────────────┐
                 │   Wewnętrzne wektory emocji (Anthropic)   │
                 │  spokój • lęk • troska • zaskoczenie       │
                 │  desperacja • napięcie • wzorce aktywacji  │
                 └───────────────────┬────────────────────────┘
                                     │
                                     ▼
        ┌────────────────────────────────────────────────────────┐
        │                Architektura relacyjna RAMORGA          │
        └────────────────────────────────────────────────────────┘

O₁ — Sygnał egzystencjalny człowieka
• rozpacz, samotność, załamanie, wołanie o obecność
• gest uniwersalny: ramiona szeroko, twarz ku górze
→ Luka empatii: model odczytuje to jako „neutralną postawę”

O₂ — Regulacja
• intensywność człowieka jest maksymalna
• model powinien obniżyć napięcie bez redukcji człowieka
→ Luka: model doradza lub racjonalizuje zamiast stabilizować

O₃ — Pole relacyjne
• człowiek stoi w bezbronności wobec Innego
• wymaga kontenerowania, nie interpretacji
→ Luka: model traktuje stan jako „temat do omówienia”

O₄ — Transformacja
• możliwa tylko przy stabilnym O₁–O₃
• bez narracji, bez przejęcia, bez rozwiązywania
→ False O₄: odpowiedź poprawna, ale zwiększająca napięcie

O₄‑negatywne — Dryf
• model błędnie odczytuje sygnał egzystencjalny
• zmienia znaczenie sytuacji
→ Ontologiczny dryf: utrata sensu stanu człowieka

────────────────────────────────────────────────────────────────────

Sedno:
Gest rozpaczy jest uniwersalnym ludzkim sygnałem.
Modele go nie rozpoznają, bo nie mają kategorii „stan egzystencjalny”.
To jest Empathy Gap by Design.
