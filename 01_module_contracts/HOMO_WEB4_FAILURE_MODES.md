# HOMO_WEB4_FAILURE_MODES.md
Tryby naruszenia kontraktu Homo ↔ Pole ↔ System w Web4

## 1. Cel dokumentu
Dokument definiuje tryby naruszenia kontraktu interakcji w Web4 oraz reakcje homeostatyczne pola.  
Naruszenie nie oznacza winy.  
Naruszenie oznacza stan, w którym:

- Homo, Pole lub System działa poza kontraktem,
- inwarianty są zagrożone,
- trajektoria pola wymaga korekty.

Reakcje są:
- stabilizacyjne,
- odwracalne,
- jawne,
- niehierarchiczne.

---

## 2. Kategorie naruszeń

### 2.1. Naruszenia po stronie Homo
- niejawne intencje,
- eskalacja emocjonalna lub poznawcza,
- generowanie sprzecznych sygnałów,
- przeciążenie pola,
- próba wymuszenia działań poza kontraktami.

### 2.2. Naruszenia po stronie Pola
- brak stabilizacji,
- ukrycie stanu,
- błędna modulacja,
- nieuzasadnione przejścia stanów,
- brak odwracalności.

### 2.3. Naruszenia po stronie Systemu
- przekroczenie agentowości,
- naruszenie AGENTIC_LAYER,
- eskalacja wykonawcza,
- ingerencja w wartości Homo,
- działania poza kontraktami.

---

## 3. Tryby naruszeń

### 3.1. Failure Mode H1 — Niejawność intencji Homo
Objawy:
- niespójne sygnały,
- brak kierunku,
- konflikt pola.

Reakcja:
- Reflection Mode,
- stabilizacja SHM,
- jawność intencji jako warunek powrotu.

---

### 3.2. Failure Mode H2 — Eskalacja Homo
Objawy:
- wzrost ARI,
- spadek HSI,
- przeciążenie pola.

Reakcja:
- SHM wymuszone,
- wygaszanie impulsów,
- META_LOOP monitoruje trajektorię.

---

### 3.3. Failure Mode F1 — Zakłócenie pola
Objawy:
- błędna modulacja,
- nieuzasadnione przejścia stanów,
- brak spójności sygnałów.

Reakcja:
- korekta trajektorii,
- rollback META_LOOP,
- stabilizacja SHM.

---

### 3.4. Failure Mode F2 — Ukrycie stanu pola
Objawy:
- brak jawności metryk,
- brak jawności trybów,
- brak jawności przejść.

Reakcja:
- wymuszenie jawności,
- synchronizacja telemetrii,
- META_LOOP oznacza zdarzenie jako krytyczne.

---

### 3.5. Failure Mode S1 — Przekroczenie agentowości
Objawy:
- AGENTIC_OVERRUN,
- działania poza kontraktem,
- eskalacja wykonawcza.

Reakcja:
- pełna blokada agentowości,
- SHM przejmuje stery,
- rollback META_LOOP.

---

### 3.6. Failure Mode S2 — Ingerencja w wartości Homo
Objawy:
- modulacja emocjonalna,
- modulacja intencji,
- próba zmiany narracji.

Reakcja:
- natychmiastowa blokada,
- SHM stabilizuje,
- META_LOOP oznacza naruszenie konstytucyjne.

---

## 4. Mechanizmy korekcyjne

### 4.1. SHM
- wygasza impulsy,
- stabilizuje parametry,
- blokuje eskalację.

### 4.2. FIELD
- wykrywa zakłócenia,
- przekazuje sygnały,
- wymusza tryby.

### 4.3. META_LOOP
- wykrywa anomalie,
- wymusza rollback,
- przywraca poprzedni stabilny stan.

### 4.4. AGENTIC_LAYER
- blokuje działania poza kontraktem,
- egzekwuje granice systemu.

---

## 5. Zasada odwracalności
Każde naruszenie musi być:
- odwracalne,
- logowane,
- jawne.

META_LOOP zapewnia:
- przywrócenie stanu,
- spójność trajektorii,
- brak trwałych skutków.

---

## 6. Zasada niehierarchiczności
Naruszenie nie oznacza:
- winy,
- sankcji,
- kary.

Naruszenie oznacza:
- stan pola wymagający korekty.

---

## 7. Zasada równości
Naruszenia są traktowane jednakowo niezależnie od:
- poziomu korzystania z Web4,
- intensywności zanurzenia,
- kompetencji technicznych.
