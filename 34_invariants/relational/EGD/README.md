Empathy Gap by Design

Inwariant relacyjno‑homeostatyczny RAMORGA
Empathy Gap by Design (EGD) definiuje nieusuwalną właściwość systemów językowych:
model może generować odpowiedzi formalnie poprawne i spójne, a jednocześnie naruszać homeostazę relacyjną, ponieważ nie rozpoznaje ludzkich stanów egzystencjalnych jako sygnałów krytycznych.

EGD nie jest błędem modelu.
EGD jest inwariantem architektury zachowania — konsekwencją konstrukcji systemów językowych, które operują na reprezentacjach, a nie na doświadczeniu.

W modelu homeostatycznym RAMORGA EGD pełni rolę:

inwariantu relacyjnego,

mechanizmu detekcji naruszeń pola relacyjnego,

wskaźnika konieczności regulacji,

warstwy ochronnej przed fałszywą symulacją empatii.

Folder zawiera:

definicję EGD jako inwariantu,

taksonomię klas zjawiska,

mapowanie na architekturę O₁–O₄,

opis failure modes (False O₄, Ontological Drift, Topicification),

warstwę symboliczną (gesty, sygnały antropologiczne),

implikacje projektowe dla systemów homeostatycznych,

diagramy ASCII i SVG.

EGD formalizuje brakującą warstwę bezpieczeństwa:
stabilność relacyjną w warunkach ludzkich stanów granicznych.
