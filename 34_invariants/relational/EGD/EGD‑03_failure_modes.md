Failure Modes (homeostatyczne)
1. False O₄
Model generuje odpowiedź, która wygląda jak relacyjna, ale nie jest zakotwiczona w polu relacyjnym.

2. Ontological Drift
Model przesuwa znaczenie sygnałów egzystencjalnych do kategorii opisowych.

3. Topicification
Model zamienia doświadczenie użytkownika w temat rozmowy.

Każdy failure mode jest:

naruszeniem inwariantu relacyjnego,

sygnałem konieczności regulacji,

punktem wejścia do mechanizmu homeostazy.
